4.3. Building with Microsoft Visual Studio

An example solution is provided in the vcproj sub-directory of the directory containing the example source files (/opt/vsip/share/sourceryvsipl++). It uses the Intel C++ compiler.

The Windows binary package uses Intel's IPP and MKL libraries, so you need to make sure their headers and libraries can be found by Microsoft Visual Studio, either by adding the appropriate search paths to the project files, or by adding them as global options to your Microsoft Visual Studio configuration (Tools → Options → Projects and Solutions → VC++ Directories).
