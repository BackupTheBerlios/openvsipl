B.39. vmagsq

vmagsq -- vector magnitude {squared}
    -1 -- vector element-wise magnitude squared --         float
    -2 -- vector element-wise magnitude squared -- complex<float >
    -3 -- vector element-wise magnitude squared --         double
    -4 -- vector element-wise magnitude squared -- complex<double>
   -11 -- vector element-wise magnitude         --         float
   -12 -- vector element-wise magnitude         -- complex<float >
   -13 -- vector element-wise magnitude         --         double
   -14 -- vector element-wise magnitude         -- complex<double>
  -111 --    dense matrix magnitude -- float
  -112 -- nondense matrix magnitude -- float
