B.29. sfilt

sfilt -- Separable_filter
   -1 -- float (min_zeropad)

Parameters:
   -p:m M       -- set filter size M (default 3)
   -p:n N       -- set filter size N (default 3)
   -p:mn MN     -- set filter sizes M and N at once
   -p:rows ROWS -- set image rows (default 16)
