B.37. vdiv

vdiv -- vector divide
                F  - float
                CF - complex<float>
   -1 -- vector element-wise divide -- F/F
   -2 -- vector element-wise divide -- CF/CF
   -5 -- real / complex vector element-wise divide
  -11 -- scalar-view vector divide (Scalar / View) -- F/F
  -12 -- scalar-view vector divide (Scalar / View) -- F/CF
  -13 -- scalar-view vector divide (Scalar / View) -- CF/CF
  -14 -- scalar-view vector divide (View / Scalar) -- F/F
  -15 -- scalar-view vector divide (View / Scalar) -- CF/CF
  -21 -- vector element-wise divide (using domains) -- F/F
  -22 -- vector element-wise divide (using domains) -- CF/CF
  -31 -- vector element-wise divide (in place) -- F/F
  -32 -- vector element-wise divide (in place) -- CF/CF
